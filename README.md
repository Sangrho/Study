# Study_link

## 목적 </br>
공부 목적으로 각 어플리케이션 별 링크들을 한 곳에 모아 놓는다. </br>

### 1차 목표 </br>
기존에 모아 놓은 사파리 북마크를 모두 옮겨 놓는다. </br>

### 2차 목표 </br>
모아 놓은 링크들을 카테고리별로 나눈다.</br>
예를 들어, Spark 에서 Memory 부분, Shuffle 부분 등을 나눈다.</br>

### 3차 목표 </br>
